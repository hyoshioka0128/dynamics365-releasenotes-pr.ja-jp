# <a name="rf-dynamic-load-building"></a>RF 動的積荷構築

| ---------- | :----------: |:----------: |

## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
この機能を使用すると、モバイル ウェアハウス RF スキャナーを使用して積荷を作成できます。 ユーザーは、WMA アプリを実行している RF スキャナーを使用して、ステージングされた品目から積荷を作成できます。 
<!--feature detail end -->
