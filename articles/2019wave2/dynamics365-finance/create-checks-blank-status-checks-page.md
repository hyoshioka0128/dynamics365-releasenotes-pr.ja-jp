## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
**小切手**ページでは、新しい小切手番号の作成や小切手の削除など、小切手のメンテナンス作業を実行します。 この機能が有効になっていると、支払プロセス中に空白状態の小切手を作成できません。 この機能強化により、小切手が不必要に無駄になるのを防ぐのに役立ちます。
<!--feature detail end -->
