## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
合理化されたデータ入力とナビゲーションにより、従業員レコードを最新の状態に保つことがより簡単になります。 将来のためのデータを入力したり、データを遡及的に更新したりできることで、今後の新入社員や退職者に関する正確なデータを提供するために必要な柔軟性が得られます。
<!--feature detail end -->
