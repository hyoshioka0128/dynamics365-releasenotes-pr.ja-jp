# <a name="update-bank-balances-when-posting-transactions-using-advanced-ledger-entry"></a>詳細な元帳エントリを使用してトランザクションを転記するときに銀行残高を更新する
[!include[finance-operations-crossapp-capabilities banner](../includes/finance-operations-crossapp-capabilities.md)]

| 有効対象    |  パブリック プレビュー | 一般提供 | 
| ---------- | :----------: |:----------: |
|エンド ユーザー、管理者/作成者による有効化、またはアナリスト|2020 年 2 月| 近日発表|


## <a name="business-value"></a>ビジネス バリュー
<!-- bv start -->
銀行口座が主勘定に関連付けられている場合、その銀行口座の残高は、関連付けられている主勘定に詳細な元帳エントリが転記されたときに更新できます。 
<!-- bv end -->



## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
この機能を使うと、銀行口座を 1 つ以上の主勘定に関連付けることができます。 銀行口座には、口座番号と支店コードが関連付けられている必要があります。
<!--feature detail end -->
