## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
複数のチーム メンバーがボットを構築している顧客サービス組織では、ボット コンテンツ資産を再利用できることが重要です。
 
会話トピックをインポートおよびエクスポートできると、ボット間でトピックを簡単に再利用および共有でき、ボット コンテンツのコラボレーションと転送が容易になります。
<!--feature detail end -->
