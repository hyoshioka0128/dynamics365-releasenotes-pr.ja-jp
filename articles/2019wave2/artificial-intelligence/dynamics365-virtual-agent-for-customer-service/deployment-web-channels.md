## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
ボットの作成者は、ボットをデモ Web サイトに配置し、カスタマー サポート組織内でそれを共有できます。 デモ Web サイトにより、独自の Web サイトを作成せずに、チーム メンバーがボットの動作を試してフィードバックを提供できます。
