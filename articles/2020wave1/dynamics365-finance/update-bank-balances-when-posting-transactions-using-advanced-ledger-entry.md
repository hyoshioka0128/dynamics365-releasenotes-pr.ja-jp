## <a name="business-value"></a>ビジネス バリュー
<!-- bv start -->
この機能により、銀行口座を 1 つ以上の主勘定に関連付けて、組織とビジネスの状況に最適な方法で口座を柔軟に整理できます。
<!-- bv end -->



## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
銀行口座には、口座番号と支店コードが関連付けられている必要があります。
<!--feature detail end -->
