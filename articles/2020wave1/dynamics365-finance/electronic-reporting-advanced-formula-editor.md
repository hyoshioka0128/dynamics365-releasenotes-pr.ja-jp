## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
電子申告の数式エディターに加えて高度な電子申告の数式エディターを使用して、電子申告 (ER) の式の構成エクスペリエンスを改善できます。 高度なエディターはブラウザー ベースで、Monaco エディターによって強化されています。 これにより、数式エディターで次の新機能を使用できます。

- コードのオートフォーマット
- IntelliSense
- コードの完了
- コードのナビゲーション
- コードの構築
- 検索と置換
- データの貼り付け
- 構文の色付け
<!--feature detail end -->
