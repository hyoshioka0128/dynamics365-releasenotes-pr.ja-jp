## <a name="feature-details"></a>機能の詳細
<!--feature detail start -->
高度な構成オプションでは以下が提供されます。

- 統一設定をリセットする機能。
- 統一された顧客活動タイムラインの説明での、検索式のサポート。
- 統一された顧客活動タイムラインに対する URL サポート。
- マージされたプロファイルに、関連する属性値を追加する機能。
